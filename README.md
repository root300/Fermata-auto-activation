# 🚗 Fermata Auto Activation (Android 14, 15, 16)

[![Support on Boosty](https://img.shields.io/badge/Support_on-Boosty-orange?style=for-the-badge&logo=boosty&logoColor=white)](https://boosty.to/fermata)
[![GitHub stars](https://img.shields.io/github/stars/root300/Fermata-auto-activation?style=for-the-badge)](https://github.com/root300/Fermata-auto-activation)

Scripts and documentation for Android Auto multimedia integration. This project provides a way to enable full multimedia access without root.

---

## 🇺🇸 English Instructions

### 🌟 Features:
* **YouTube & Web Browser** directly on your car's screen.
* **No Root** required.
* Full support for **Android 14, 15, and 16**.

### 🚀 How to get access:
1. Click the **[Support on Boosty](https://boosty.to/fermata)** button above.
2. Enter **$5** (approx. 500 RUB).
3. **IMPORTANT:** You must include your **Gmail address** in the message field.
4. After verification, you will receive instructions via email.

## 🇷🇺 Инструкция на русском

### 🌟 Возможности:
* **YouTube и Браузер** на экране вашего автомобиля.
* **Без Root-прав**.
* Полная поддержка **Android 14, 15 и 16**.

### 🚀 Как получить доступ:
1. Нажмите кнопку **[Поддержать на Boosty](https://boosty.to/fermata)** выше.
2. Введите сумму **$5** (около 500 РУБ).
3. **ВАЖНО:** Обязательно укажите ваш **Gmail** в поле для сообщения.
4. После подтверждения вы получите инструкции на указанную почту.
